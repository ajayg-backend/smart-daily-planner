# 🚀 TaskFlow (Daily Planner)

A simple daily planner web app built using Flask.

## ✨ Features
- Add tasks with time
- Popup alert system
- Sound notification
- Delete tasks

## 🛠 Tech Used
- Python (Flask)
- HTML, CSS
- JavaScript
